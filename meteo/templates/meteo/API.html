Radar do Tempo

Documentação da API

Endpoint: /meteo/cidades

Descrição: Retorna uma lista de cidades e suas informações meteorológicas.

Exemplo de Uso:

curl -X GET https://a22205897.pythonanywhere.com/meteo/cidades

Exemplo de Resposta:

{
  "pais": "Portugal",
  "regioes": [
    {
      "local": "Lisboa",
      "globalIdLocal": 1110600,
      "color": "green"
    },
    {
      "local": "Porto",
      "globalIdLocal": 1131200,
      "color": "red"
    },
    ...
  ]
}

A resposta contém uma lista de regiões (cidades) em Portugal, cada uma com seu nome, ID global e cor de alerta associada.

Endpoint: /meteo/previsao/hoje/{globalIdLocal}

Descrição: Retorna a previsão meteorológica para hoje para uma cidade específica.

Exemplo de Uso:

curl -X GET https://a22205897.pythonanywhere.com/meteo/previsao/hoje/{globalIdLocal}

Exemplo de Resposta:

{
  "cidade": "Lisboa",
  "previsoes": [
    {
      "data": "2024-06-07",
      "temperatura_maxima": 25,
      "temperatura_minima": 15,
      "precipitacao": 30,
      "id": 1,
      "descricao": "Céu limpo"
    }
  ]
}

A resposta contém a previsão meteorológica para hoje para a cidade especificada, incluindo a data, temperatura máxima e mínima, probabilidade de precipitação, ID e descrição do tipo de tempo.

Endpoint: /previsao/img

Descrição: Retorna o link da imagem do tempo de hoje em Lisboa.

Exemplo de Uso:

curl -X GET https://a22205897.pythonanywhere.com/previsao/img

Exemplo de Resposta:

{
    "src": "https://a22205897.pythonanywhere.com/static/meteo/w_ic_d_03anim.svg"
}
